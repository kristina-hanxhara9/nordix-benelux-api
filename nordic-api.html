✓ Free — No account, no key Norway — BRREG Enhetsregisteret

Best-in-class. Fully open REST API, no registration needed, bulk download included. Up to 3 NACE codes per company, employee count, and VAT status natively included in every response. Licence: NLOD (Norwegian open government data).

Free — Credentials required (email for free access) Denmark — CVR Virk

Elasticsearch-based API (POST queries with JSON body). Richest data of all Nordics — includes company purpose text, phone, email, website, and full ownership history. 2.2M+ companies. Contact cvrselvbetjening@erst.dk for free credentials.

http://distribution.virk.dk/cvr-permanent/virksomhed/_search

✓ Free — No account, no key Finland — PRH / YTJ open data

Free REST API, no authentication. Unique bonus: digital financial statements (P&L + balance sheet) via iXBRL for companies that file digitally. Full bulk JSON download of all companies available. Licence: CC BY 4.0.

Mixed — SCB free, Bolagsverket requires contract Sweden — SCB (free) + Bolagsverket (paid)

Two sources. Bolagsverket (official registry) requires a contract + SEK 6,250 upfront fee. Statistics Sweden (SCB) is free and includes SNI/NACE codes and monthly updated CSVs per canton. Use SCB for bulk free work, Bolagsverket for directors/financials.

https://data-bs.ch/stata/zefix_handelsregister/all_cantons/companies_ZH.csv
