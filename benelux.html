Benelux Company Registry APIs

Free vs paid breakdown · Updated March 2026 · BE · NL · LU

€6.40/month subscription required Free calls on Zoeken & Financial Statements Netherlands — KVK Handelsregister

The subscription fee (€6.40/month) unlocks all four APIs. Zoeken is then free per call and returns company names. Basisprofiel costs €0.02/call on top. There is also a truly free open dataset for financial statements — but it is anonymised, meaning the company name is deliberately removed.

📢 2026: UBO API expected — will expand beneficial ownership access to those with “legitimate interest”. Not yet live as of March 2026.
Free per call (subscription key required) 1. Zoeken API — Search

Use to search by name, postcode, city, or entity type. Returns company name and a link to fetch the full profile. Max 1,000 results per query, 100 per page.

Search parameters
naam — company name (full text match only)
postcode + huisnummer — postcode + house number
plaats — city
type — see entity types below
kvkNummer — direct lookup
rsin, vestigingsnummer — ID lookups
Entity types (type=)
hoofdvestiging — main branch (primary registered location)
nevenvestiging — branch office (secondary locations)
rechtspersoon — legal entity (the company as a legal person)

Combine: ?type=hoofdvestiging&type=rechtspersoon
Tip: use hoofdvestiging or rechtspersoon to avoid duplicate hits when a company has many branches.
Returns in each result
+naam — company name (statutory name or first trade name) ✓ names ARE returned
+kvkNummer, vestigingsnummer, rsin
+Address (street, city, postcode if searched by it)
+type (hoofdvestiging / nevenvestiging / rechtspersoon)
+actief — whether active or deregistered
+Links to Basisprofiel and Vestigingsprofiel for chaining
−SBI code — NOT a search filter, NOT returned here
−Employee count, financials, shareholders — not here
https://api.kvk.nl/api/v2/zoeken?naam=Philips&type=hoofdvestiging
https://api.kvk.nl/api/v2/zoeken?postcode=1234AB&huisnummer=5
⚠ Search is exact-match only, not fuzzy. “koophand” returns zero results; “koophandel” returns ~1,000. No SBI filter exists on any KVK API.
€0.02 per call 2. Basisprofiel API — Full Company Profile

Called with a KVK number (from Zoeken). Returns everything about a company in one response.

Returns
+Full name, KVK number, RSIN, legal form
+SBI code(s) + description — this is the only place SBI appears with a company name
+Full registered address with geocoordinates (BAG)
+Registration date
+Full-time + part-time employee counts
+Website
+Main owner / shareholder name (one only)
+Branch list with links
−Full shareholder list (private register)
−Directors / officers (use Vestigingsprofiel)
−Financials (use free Financial Statements dataset)
https://api.kvk.nl/api/v1/basisprofielen/68750110
Truly free — no subscription needed 3. Open Dataset Financial Statements — Anonymised

Free, CC BY 4.0, no account. Returns balance sheet, income statement, cash flow for companies that file accounts digitally via XBRL. Deliberately anonymised — KVK strips the company name and KVK number because for sole traders and small BVs the name is personal data.

To use this meaningfully: call Basisprofiel first (paid) to get the KVK number and name, then query this endpoint with that KVK number to attach financials. The two datasets are designed to be joined — just not for free in bulk with names attached.
Returns (XBRL filers only)
+SBI code
+Financial year + filing date
+Balance sheet: total assets, fixed assets, current assets, equity, liabilities
+Income statement: revenue, result for the year, operating result
+Cash flow statement (where applicable)
−Company name — intentionally removed
−KVK number — intentionally removed from the response
https://opendata.kvk.nl/api/v1/hvds/jaarrekeningen/kvknummer/68750110
Free — bulk CSV via SFTP (email registration) Belgium — CBE / KBO

One access model: free bulk CSV. There is also a paid SOAP API (€50 / 2,000 requests) but it returns exactly the same data as the free CSV, just per-query in real time. For data science purposes the free CSV is always the right choice — there is no premium tier with extra fields.

📢 June 2025: Data updated daily (was Tue–Sat). Jan 2026: all 7-digit NACE codes auto-replaced with 5-digit equivalents across all active entities.
Free Open Data CSV (SFTP) — everything that exists
All fields available (spread across CSV files in the ZIP)
+enterprise.csv — EnterpriseNumber (10-digit), Status, JuridicalSituation, legal form, StartDate, TypeOfEnterprise
+denomination.csv — company name in Dutch, French, German
+activity.csv — NACE-BEL 2025 codes (5-digit, multiple per company)
+address.csv — full street address, postcode, municipality
+contact.csv — phone, email, website
+function.csv — directors and legal representatives (name, role, start date)
+establishment.csv — branch / establishment units
−Shareholders — not in CBE at any tier
−Financial statements — not in CBE at any tier

Free portal — PDFs + purpose text Paid professional API No self-service signup Luxembourg — LBR / RCS + RBE

Three access tiers. The professional API (since 2022) provides structured data including NACE codes, shareholders, and purpose text as a field. The free portal has the same purpose text in HTML — scrapeable. No emails or websites at any tier — the registry only stores legal data. 168,400+ entities as of December 2025.

📢 Jan 2026: LBR announced major data quality reforms — automated + manual filing checks, fines for late filing (€50–€500). NACE codes now formally part of RCS via monthly STATEC collaboration.
Free — no account Tier 1 — Free Public Portal (lbr.lu)
Available free — portal search or HTML scrape
+Company name + RCS number
+Legal form (SA, Sarl, SCS, SCSp, SE etc.)
+Registered address
+Share capital
+Purpose text (objet social) — shown in HTML on company page, scrapeable. Use for AI-based NACE classification. Typically very specific in Luxembourg filings.
+Annual accounts — PDF, free, last 20 years
+Articles of association — PDF, free
+RESA publication history (official gazette)
−NACE code — only in professional API
−Shareholders — only in professional API
−Website or email — not collected by LBR at any tier
−Bulk download — not available
💡 For NACE without paying: scrape the purpose text from the free portal → run your own classifier. Most LU entities cluster into 4–5 purpose patterns (holding, fund, real estate, services). Simple keyword matching covers ~70% of the corpus; an LLM call handles the rest. At ~€0.001/call that’s roughly €40 for the full 168K.
https://www.lbr.lu
https://www.lbr.lu/mjrcs-resa/
Paid — LuxTrust + signed agreement Tier 2 — Professional API (structured JSON)

Available since 2022. Targeted at fiduciaries, law firms, KYC platforms. Contact helpdesk@lbr.lu. No public pricing.

All Tier 1 data, structured as JSON, plus:
+NACE code — pre-assigned monthly via LBR × STATEC
+Purpose text as a structured field (same content as portal, but JSON)
+Shareholders — name, address, shares held, share type
+Directors — name, role, signatory powers (individual / joint)
+Registration history and amendment events
+Insolvency register (REGINSOL) cross-reference
−UBO data — Tier 3 only
−Website / email / phone — not in registry at any tier
