Exact filter parameters, use-cases, and example API calls for each country · March 2026

Free · No auth 🇳🇴 Norway — BRREG API

Most flexible free registry in Europe. Query by almost anything. All filters combinable. Real-time, live data.

Free · Email credentials required 🇩🇰 Denmark — CVR Elasticsearch API

Elasticsearch backend means you can filter on almost any field in any combination. Most powerful query model of all seven countries — but requires Elasticsearch syntax (POST with JSON body).

No live API — bulk CSV only 🇸🇪 Sweden — SCB / Bolagsverket

Sweden has no free company-level lookup API. The free route is a monthly CSV from SCB. All filtering must be done offline after downloading.

⚠ You cannot query Sweden's registry live by name, NACE, or status — you download the whole dataset and filter it yourself.

€6.40/month subscription 🇳🇱 Netherlands — KVK Handelsregister

Zoeken is free per-call (subscription key needed). Limited filter params — notably no SBI code filter on the live API. For SBI filtering you must use the offline open dataset or a third-party.

⚠ You CANNOT filter by SBI code on any KVK API endpoint. This is the biggest limitation vs the Nordic registries.

Free bulk CSV (daily) 🇧🇪 Belgium — CBE / KBO

Belgium has no live query API in the free tier. You download the full CSV daily and filter offline. This is actually fine — the CSV is complete, fast to process, and has NACE-BEL codes, directors, contacts, and status all together.
